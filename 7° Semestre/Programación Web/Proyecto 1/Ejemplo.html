<BR>


La etiqueta < br > en HTML se utiliza para crear un salto de línea en el contenido, lo que significa que el texto o los elementos que vienen después de <br> se mostrarán en la línea siguiente. Es útil cuando quieres separar visualmente elementos en tu página web sin necesariamente crear un nuevo párrafo. Por ejemplo, puedes usar <br> para separar líneas de una dirección o para dar espacio entre líneas de texto.

Ejemplo

Hola,
<br > es un salto de línea.

<a>

La etiqueta <a> en HTML se utiliza para crear enlaces a otras páginas web, archivos, direcciones de correo electrónico, ubicaciones en la misma página, entre otros. Su atributo principal es href, que especifica la URL a la que se debe dirigir el enlace.

Ejemplo de uso de la etiqueta <a> para crear un enlace a una página web:

<a href="https://www.ejemplo.com">Visitar ejemplo.com</a >

<img>

La etiqueta <img > en HTML se utiliza para incrustar imágenes en una página web.
