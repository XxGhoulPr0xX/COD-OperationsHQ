Selectores

1. Selector de Elemento

Selecciona todos los elementos que coinciden con un nombre de etiqueta específico.

h1 { color: blue; }

2. Selector de Clase

Selecciona todos los elementos que tienen un atributo de clase específico.

.destacado { font-weight: bold; }

3. Selector de ID

Selecciona un elemento con un atributo de ID específico.

#encabezado { background-color: gray; }

4. Selector Universal

Selecciona todos los elementos en el documento.

* { margin: 0; padding: 0; }

5. Selector de Atributo

Selecciona elementos que tienen un atributo específico.

[type="text"] { border: 1px solid black; }

6. Selector de Combinación

Selecciona elementos basados en su relación con otros elementos.

p + p { margin-top: 10px; }

7. Selector de Pseudo-clase

Selecciona elementos en función de su estado o posición en relación con el documento.

a:hover { text-decoration: underline; }

8. Selector de Pseudo-elemento

Selecciona partes específicas de un elemento que no están representadas por clases o IDs.

p::first-line { font-weight: bold; }
